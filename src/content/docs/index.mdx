## How to Use This Site

1. **Follow the recommended order** — Modules build on each other, starting with foundations and ending with practical integration patterns.
2. **Budget ~12 hours total** — Each module lists estimated study time per section.
3. **Use the resources** — Every section links to articles, videos, papers, and documentation. Read/watch the ones that interest you most.
4. **Reflect & Apply** — Each module ends with questions to help you connect the material to your role.
5. **Search freely** — Use the search bar (top of page) to find any topic across all modules.
6. **Check the [Glossary](./glossary/)** — Key terms defined and linked back to the relevant module.
